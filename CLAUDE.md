# DMBOK AI — Claude Code 向け指示

このファイルは Claude Code がプロジェクトを理解するためのエントリーポイントである。
プロジェクト全体の構成と方針は `AGENTS.md` を参照。

## スキル

以下の2つのスキルが利用可能。ユーザーの依頼に応じて該当するスキルの README.md を読み、指示に従うこと。

### DMBOK 診断（アセスメント）

**トリガー**: 「DMBOK診断して」「成熟度を評価して」「リポジトリを分析して」「データ管理の現状を知りたい」など

→ `skills/dmbok-assess/README.md` を読んで実行する

### 壁打ち・相談（コンサルティング）

**トリガー**: 「データガバナンスについて相談したい」「データ管理をどうすればいい」「DMBOKって何」など

→ `skills/dmbok-consult/README.md` を読んで実行する
